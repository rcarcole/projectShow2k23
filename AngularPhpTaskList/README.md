## Frontend

### Installation

To install and run this project locally, follow these steps:

```
git clone https://github.com/rcarcole/projectShow2k23.git
cd AngularPhpTaskList
npm install
ng serve
```

### Use

After running `ng serve`, visit `http://localhost:4200/` in your browser.
